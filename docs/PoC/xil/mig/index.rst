
mig
===

The namespace ``PoC.xil.mig`` offers pre-configured memory controllers generated
with Xilinx's Memory Interface Generator (MIG).

* **for Spartan-6 boards:**
 
  * :doc:`mig_Atlys_1x128 </PoC/xil/mig/mig_Atlys_1x128>` - A DDR2 memory controller for the Digilent Atlys board.
 
* **for Kintex-7 boards:**
 
  * :doc:`mig_KC705_MT8JTF12864HZ_1G6 </PoC/xil/mig/mig_KC705_MT8JTF12864HZ_1G6>` - A DDR3 memory controller for the Xilinx KC705 board.
